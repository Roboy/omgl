### omgl ###
helper classes for opengl rendering

### dependencies ###
eigen3, sfml, glew, opencv, boost, pcl, opengl, assimp, sdformat


### build and run ###
```
#!bash
cd path/to/omgl
mkdir build
cd build
cmake ..
make 
```
you will need to edit the rootdirectory path in main.cpp:
```
#!bash
cd path/to/omgl/bin
./omgl
```
